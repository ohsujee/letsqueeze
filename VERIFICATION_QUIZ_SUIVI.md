## REGLES DE VERIFICATION

### A verifier pour chaque question:
1. **Reponse dans la question?** - Le nom/mot de la reponse ne doit PAS apparaitre
2. **Spoiler logique?** - Pas de deduction immediate (ex: "le frere de X")
3. **Veracite** - Recherche web multi-sources pour confirmer les faits
4. **Pyramidalite** - Du plus obscur au plus evident
5. **Phrases fluides** - Pas de style telegraphique
6. **VO/VF** - Pour series/films/anime, mettre les deux versions si differentes

### Format reponse VO/VF:
```
"answer": "Nom VF (VO: Nom Original)"
```
ou si le nom VO est plus connu:
```
"answer": "Nom VO (VF: Nom Francais)"
```
